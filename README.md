# Marly Escritorio

Sistema de gestión para tienda de ropa, diseñado para optimizar el control de inventario y ventas.

## Módulos del Sistema

*   **Ventas (Punto de Venta)**: Interfaz ágil para registrar salidas de mercancía, cobros y cambios de estado (venta, apartado, crédito).
*   **Inventario**: Vista general de existencias con filtrado por categoría, talla y estado. Permite auditorías y ajustes manuales.
*   **Entradas**: Registro de nueva mercancía y reabastecimiento de proveedores.
*   **Estadísticas**: Tableros de análisis para visualizar rendimiento diario, productos más vendidos y rotación de stock.

## Tecnología

Construido con una arquitectura moderna y eficiente:
*   **Electron**: Para la experiencia de escritorio nativa.
*   **React + Vite**: Para una interfaz de usuario rápida y reactiva.
*   **TypeScript**: Para un código robusto y seguro.

## Instalación y Uso

1.  Instalar dependencias: `npm install`
2.  Modo desarrollo: `npm run dev`
3.  Construir para producción: `npm run build`
